# Crud Forum Laravel

## Requisitos
- PHP/Servidor PHP
- [Composer](https://getcomposer.org/)

## Configurações
Configurando Ambiente:
> Encontra-se no arquivo: .env

## Iniciando
Instalar dependências:
> Todos comandos executador na pasta raiz do projeto ("forum\") via terminal

`composer install` 

`npm install && npm run dev` 

## Database
Migrations:
`php artisan migrate` 

## Subindo projeto
`php artisan serve`
